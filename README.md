# GBMarketWatch
Version 0.0.1-beta
THIS SCRIPT IS STILL IN DEVELOPMENT AND NOT YET MEANT TO BE USED IN PRODUCTION

## Requirements
nodejs & npm

## Install
- git clone or download the .zip and extract in your GB folder
- run: npm install
