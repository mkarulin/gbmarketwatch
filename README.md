# GBMarketWatch
Version 0.0.1-beta
THIS SCRIPT IS STILL IN DEVELOPMENT - USE AT OWN RISK

## Requirements
nodejs & npm

## Install
- git clone or download the .zip
- run: npm install
- run: node index.js
